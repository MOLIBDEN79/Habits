# Скриншоты

Положи сюда PNG-файлы со следующими именами — они автоматически появятся в главном README:

| Файл | Что снять |
|------|-----------|
| `home.png` | Главная страница (http://localhost:8080/) |
| `profile.png` | Профиль (http://localhost:8080/profile) |
| `calendar.png` | Календарь (http://localhost:8080/calendar) |
| `login.png` | Страница входа (http://localhost:8080/login) |

**Как снять:** запусти приложение, открой страницу, сделай скриншот (`Win + Shift + S`),
сохрани с нужным именем в эту папку, затем `git add docs/screenshots && git commit && git push`.
